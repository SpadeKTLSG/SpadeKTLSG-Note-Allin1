### 描述目录-文件操作

例如, 在何处写入文件与修改

‍

‍

增

‍

 **&gt;&gt;** ​​

在目标地增加文件

```java
resource.fuck >> Beach.java
```

‍

‍

删

‍

**RM**

在目标地删除文件

```java
resource.fuck  rm  Beach.java
```

‍

定位

 **-&gt;    或    |**

表示定位强调目标

```java
resource.fuck -> Beach.java
resource.fuck | Beach.java
```
